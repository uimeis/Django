### 链接数据库
setting.py将默认的`sqlite3`改成`sqlite3`，`NAME`是数据库名称，`USER`数据库的用户，`PASSWORD`是密码，`HOST`是主机地址，PORT是端口号。

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'django_test',
        'USER': 'root',
        'PASSWORD': '127.0.0.1',
        'PORT': '3306',
    }
}
```
