### if标签
views.py

```python
def index(request):
    context = {'text': '中国'}
    return render(request, 'index.html', context=context)
```

index.html

```html
<body>
    {% if text == '中国' %}
        <p>我爱你中国</p>
    {% else %}
        <p>中国，很棒</p>
    {% endif %}
</body>
```

### for..in..标签
views.py

```python
def index(request):
    context = {'text': ['中国', '日本', '韩国']}
    return render(request, 'index.html', context=context)
```

index.html

```html
<body>
    {% for i in text %}
        <p>{{ i }}</p>
    {% endfor %}
</body>
```
`for`循环中还有一些变量
* `forloop.counter`当前循环的下标，从1开始
* `forloop.counter0`当前循环的下标，从0开始
* `forloop.first`是否是第一次遍历，返回True
