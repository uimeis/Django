### 分离url
在app中可以看到views.py，分离了视图函数，我们也可以创建urls.py，分离url。
```python
from django.urls import path
from .views import book

urlpatterns = [
    path('', book, name='图书'),
]
```
如果上面叫**子url**，我们去**主url**接收子url
```python
from django.urls import include

urlpatterns = [
    path('book/', include('book.urls'))
]
```
